# WarNet-Air-Fuel
El Air-Fuel de WarNet es un Aerodirigible Exterminator, sin limitación de alcance autónomo. Puedes cruzar la estratosfera y no presenta límites autónomos. Está 2° GEN. está limitada en componentes y funcionalidad reduciendo costes con mínimas perdidas de eficiencia y mínimos requisitos necesarios.
